## Cuestionario:
¿Qué es una Base De Datos Relacional?  
Es un sistema que guarda información en tablas y permite relacionar esos datos entre sí mediante claves

¿Qué es una Tabla en el contexto Base De Datos Relacionales?  
Es una estructura que organiza los datos en filas y columnas, como una hoja de cálculo

¿Qué es una relación en el contexto de Base De Datos Relacionales?  
Es el vínculo entre dos tablas, normalmente usando claves primarias y foráneas

¿Qué representa una columna en el contexto de Base De Datos Relacionales?  
Representa un atributo o característica de los datos, como “nombre” o “precio”

¿Qué representa un renglón en el contexto de Base De Datos Relacionales?  
Representa un registro individual, por ejemplo un cliente o un producto

¿Qué es un motor de base de datos relacional?  
Es el software que administra, organiza y permite consultar la base de datos

¿Cuáles son los motores de bases de datos relacionales más populares?  
MySQL, PostgreSQL, Oracle Database, Microsoft SQL Server e IBM Db2

¿Cuál es el motor de base datos ligero que no necesita un servidor?  
SQLite

¿Qué significa SQL en el contexto de base datos relacionales?  
Structured Query Language, el lenguaje estándar para manejar y consultar datos

¿Cuáles son los elementos de un sistema de información basado en base de datos relacionales?
Datos (tablas)
Hardware (computadoras, servidores)
Software (motor de base de datos, aplicaciones)
Usuarios (administradores, analistas, clientes)
Procesos (reglas y flujos de trabajo)
